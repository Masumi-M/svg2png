# svg2png
=====
## Description
Python script to convert SVG file to PNG file.

## Requirements
- macOS Catalina (v10.15.7)
- Python v3.6.5
- pip 20.3.3

## Setup
`pip install svglib`

## Usage
`python svg2png sample.svg`
